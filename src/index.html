David Minnerly

I'm a 19-year-old programming enthusiast living in the state of New York. I aim to make a career out of server administration and network security, and in my spare time I create websites and command-line tools.

I frequently work on open-source projects. All of which can be found on my GitHub.

I got my first taste of programming when I was 13. I started out by learning Lua on the online building game ROBLOX. Making my little blocky creations come to life was amazing to me, and it became the foundation for something more.

Since then I've branched out to learning about web design, picking up on HTML, CSS and JavaScript along the way. I've experimented with creating applications on AngularJS and Django, and both of them were great fun to work with.

Projects

Fireflies: a small application that displays a quote and background image at random every time the page is loaded.

It started as an AngularJS app. It fit my needs very well at the start, but a major drawback to my setup was that new images and quotes were cumbersome to implement.

I later fixed this by porting Fireflies to Django. This works considerably better, as now when I want to add a new image or quote, I can do it directly from the admin interface.

All of the images and quotes mean a lot to me, I picked ones that I personally liked, and because of that I hold Fireflies very near to my heart.
